Privacy Policy

Last updated: June 18, 2026

What we collect

Threat Hunters only collects account details, scan targets submitted by users, scan results, support messages, and security preferences needed to operate the platform.

How we use data

We use data to authenticate users, run requested security checks, generate reports, improve product reliability, and respond to support or responsible disclosure requests.

Security posture

API keys and sensitive backend credentials are never exposed to the browser. Security checks that require protected credentials run through backend routes.

Retention

Users can remove account data from the User Console. Development environments use the same MongoDB Atlas-backed data model as the rest of the project.
