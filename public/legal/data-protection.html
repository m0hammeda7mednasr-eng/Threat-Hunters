Threat Hunters Security

Data Protection

Last updated: June 18, 2026

Backend-first secrets

Threat Hunters keeps provider keys and protected integrations on the backend. Browser code calls internal API routes instead of third-party secret-backed APIs directly.

Scan data

Scan reports are tied to the user console and can be downloaded as PDFs for remediation. Sensitive targets should only be scanned with authorization.

Account controls

The User Console supports profile updates, password changes, two-factor preference storage, report settings, and account deletion.

Operational hygiene

Use strong passwords, rotate exposed credentials, protect environment variables, and review reports before sharing them externally.
